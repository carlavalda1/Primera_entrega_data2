# Primera_entrega_data2
Tema elegido
Relación entre crecimiento económico, consumo energético y emisiones de CO₂
Industria objetivo: energía, finanzas, políticas públicas
Fuente de datos: World Bank Open API (datos públicos)
Abstracto – Motivación y Audiencia
Motivación
El crecimiento económico genera oportunidades de negocio, pero también desafíos ambientales.
Comprender cómo el consumo energético y las emisiones de CO₂ evolucionan junto al PBI permite a empresas energéticas, bancos y policymakers tomar decisiones estratégicas basadas en datos.
Audiencia

Ejecutivos del sector energético
Analistas de riesgo y sostenibilidad
Organismos de planificación económica
Equipos de data analytics

Preguntas / Hipótesis

¿Existe relación entre el crecimiento del PBI y las emisiones de CO₂?
¿El aumento del consumo energético implica necesariamente más emisiones?
¿Qué países muestran crecimiento económico con menor impacto ambiental?
¿Se observan patrones diferenciados por región?
